## Installation poetry
Windows
```power shell
    (Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | py -
```
Other os
[Instruction](https://python-poetry.org/docs/#installation)

## Инициализация проекта
```shell
poetry install
```

### Запуск сервера FLASK
```shell
flask run --debug
```
